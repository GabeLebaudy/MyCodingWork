HTML attributes:
Width: Specifies the width of a media source
Height: Specifies the height of a media source
Alt: Displays text in place of media source if the website cannot load the media Style: Changes the style of an element I.e. font, color, text size, etc...
Lang: goes in html tag, specifies language of the website for search engines, but doesn't actually affect display (en = English)
Title: The title attribute will be displayed as a tooltip when the mouse hovers over the element
